### VARCO API 플랫폼 소개
#### 상상력을 AI 기술로 연결하는 VARCO API 솔루션 

단 한 번의 연동으로 텍스트, 이미지, 오디오, 데이터 처리 등 다양한 AI 기능을 손쉽게 활용하세요. 개발자는 API문서와 예제를 통해 빠르게 통합할 수 있으며, 모든 호출 내역과 크레딧 사용을 한눈에 관리할 수 있습니다. 지금 바로 API를 호출하고, 당신의 제품에 AI 기능을 더해보세요.


### API Reference
각 API의 기술 명세를 정리한 개발자용 레퍼런스입니다. 엔드포인트, 요청 메서드, 필드 정의, 응답 스키마, 예제 코드 등 API 호출에 필요한 모든 사양을 명확히 제공합니다. 각 상품의 도큐먼트와 연결되어 있으며, 개발 중 API를 호출하거나 파라미터를 검증할 때 활용할 수 있습니다.
엔드포인트 별 최신 버전은 항상 이 페이지를 기준으로 확인하세요.

- 핵심 목적: API 스펙 및 구조 이해 중심 
<br>
- 주요 구성: 엔드포인트 목록 / 요청-응답 파라미터 / 예제 요청 / 에러 코드
